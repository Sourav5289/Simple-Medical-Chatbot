💻 This Python program is a simple yet smart medical chatbot named MediBot 🤖 designed to provide general health information.<br> It uses a predefined medical knowledge base 🩺 to respond to common symptoms like headache, fever, and cough. <br>MediBot can also recognize greetings 👋, farewells 👋, and expressions of gratitude 🙏, offering friendly and context-aware replies. <br> Using conditional logic and random responses, it creates a natural, conversational experience 💬 while reminding users to consult a doctor for serious conditions
